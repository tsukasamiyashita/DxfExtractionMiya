DxfExtractionMiya v1.0.0

DxfExtractionMiyaは、CADの図面データ（DXFファイル）からテキスト（文字）情報を抽出し、見やすいExcel形式（表形式）に自動変換するWindows向けデスクトップアプリケーションです。
また、バラバラに出力された複数のExcelファイルを1つのマスターファイルに集約（結合）する機能も備えています。

🌟 主な特徴

簡単操作: DXFファイルを選択してボタンを押すだけで、自動的にテキストを読み取ってExcel（.xlsx）ファイルとして保存します。

自動表形式化: 図面上で横並びになっている文字を自動的に判断し、Excelの同じ行のセルに分割して出力します。

ズレ許容値の調整: 「同じ行とみなすY座標のズレ」を数値で調整できるため、図面の縮尺や文字の配置に合わせた柔軟な抽出が可能です。

データの一括集約: フォルダを指定するだけで、中にある複数のExcelファイル（抽出結果など）を自動で読み込み、1つのファイルに全データをまとめて結合します。先頭列には元のファイル名が自動で付与されるため、データ管理が非常に簡単になります。

🚀 基本的な使い方

準備

アプリ（.exeファイル等）をダブルクリックして起動します。

⚠️ 【重要】初回起動時の注意点について
インターネットからダウンロードしたアプリを初めて実行した際、青い画面で**「Windows によって PC が保護されました」**という警告（SmartScreen）が表示される場合があります。
これはWindowsの標準的な保護機能であり、ウイルスではありませんのでご安心ください。
【回避手順】

警告画面の左側にある 「詳細情報」 という文字をクリックします。

右下に新しく表示される 「実行」 ボタンをクリックするとアプリが立ち上がります。

パターンA：DXFファイルからテキストを抽出する

画面上部の 「📄 DXFファイルを選択 (抽出用)」 ボタンを押します。

テキストを抽出したいDXFファイルを選びます（複数選択可能です）。

「設定」欄で、保存先を決めます。
図面上のテキストがうまく横並びにならない場合は、「同じ行とみなすY座標のズレ許容値」の数値を少し大きくして再度お試しください。

画面下部の緑色のボタン 「🚀 DXFからテキストを抽出 (Excel化)」 を押すと処理が開始されます。

パターンB：複数のExcelファイルを1つに集約する

画面上部の 「📁 フォルダを選択 (集約用)」 ボタンを押します。

まとめたいExcelファイルが入っているフォルダを選択します。

画面下部の紫色のボタン 「🧩 複数Excelを1つに集約 (結合)」 を押します。

選択したフォルダ内のExcelファイルが結合され、「データ集約_日付.xlsx」という名前で新しく保存されます。
