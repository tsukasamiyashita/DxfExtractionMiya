DxfExtractionMiya v1.3.0

## アプリの概要

DxfExtractionMiyaは、CADの図面データ（DXFファイル）からテキスト（文字）情報を抽出し、見やすいExcel形式（表形式）に自動変換するWindows向けデスクトップアプリケーションです。
複数のDXFファイルやフォルダを選択した場合は、すべての抽出結果が自動的に1つのExcelファイル（マスターファイル）に集約されて出力されます。

## 主な機能と特徴

*   **簡単操作:** DXFファイルを選択してボタンを押すだけで、自動的にテキストを読み取ってExcelファイル（.xlsx）として保存します。
*   **2つの抽出モード:**
    *   **指定範囲の抽出（プレビュー機能）:** 図面のプレビュー画面を見ながら、基準となる文字と抽出したい範囲をマウスで直接指定できます。図面の縮尺や配置ズレにも対応しており、特定の項目だけを正確に抜き出して一覧化したい場合に最適です。複数の図面を選択した場合、自動で1つのリストに集約されます。
    *   **全体抽出:** 図面上の横並びになっている文字を自動的に判断し、Excelの同じ行のセルに分割して一括出力します。「同じ行とみなすY座標のズレ許容値」の数値を調整できるため、図面の縮尺や文字の配置に合わせた柔軟な抽出が可能です。
*   **全角文字の自動変換:** 抽出されたテキストに含まれる全角英数字、記号、カタカナ、スペースなどは、自動的に半角に変換されてExcelに出力されます。図面ごとの表記の揺れを抑え、データとしての扱いやすさを向上させます。
*   **設定の保存と読み込み機能:**
    *   **環境設定の保存:** アプリのウィンドウサイズや表示内容を保存し、次回起動時に復元します。
    *   **DXF抽出設定の個別管理:** 抽出する範囲や項目の設定を個別のファイルとして保存・読み込みできます。メイン画面上で「現在開かれている設定ファイル名」や、読み込み後に変更が加えられたかを示す「(変更あり)」の表示が確認できるため、複数パターンの設定も簡単に管理できます。また、最後に開いていた設定ファイルの内容は次回起動時にも自動的に反映されます。

## 動作環境

*   **OS:** Windows 10 / Windows 11

## インストールと起動方法

1.  配布された `DxfExtractionMiya-v1.3.0.exe` をPC内の任意のわかりやすい場所（デスクトップや専用フォルダなど）に保存します（インストール作業は不要です）。
2.  `DxfExtractionMiya-v1.3.0.exe` をダブルクリックして起動します。

⚠️ **【重要】初回起動時の注意点について**
インターネットからダウンロードしたアプリを初めて実行した際、青い画面で 「Windows によって PC が保護されました」 という警告（SmartScreen）が表示される場合があります。
これはWindowsの標準的な保護機能であり、ウイルスではありませんのでご安心ください。

**【回避手順】**
1.  警告画面の左側にある 「詳細情報」 という文字をクリックします。
2.  右下に新しく表示される 「実行」 ボタンをクリックすると、アプリが正常に立ち上がります。

## 基本的な使い方

### パターンA：特定の項目を狙って抽出・リスト化する（プレビュー機能を利用）

図面内の特定のタイトルや数値などを、基準となる文字の位置から計算して抽出し、1つのExcelにまとめます。

1.  **対象の選択:**
    画面上部の「📄 DXFファイルを選択」または「📁 DXFフォルダを選択」ボタンを押し、対象を選びます（複数ファイルの一括処理が可能）。

2.  **設定:**
    *   「2. 設定」欄の「1. プレビュー画面から抽出範囲を選択して抽出」を選択します。
    *   「＋ プレビューを開いて基準文字と抽出範囲を設定」ボタンを押します。

3.  **プレビュー画面での操作:**
    *   **手順1:** 抽出の絶対的な基準となる文字（図面内で位置が変わらない文字）を左クリックします（赤色になります）。
    *   **手順2:** 必要に応じて、図面ごとの縮尺・配置補正用の第2基準文字をクリックします（任意・不要な場合はそのまま次へ）。
    *   **手順3:** 抽出したい値がある場所を、マウスの左ドラッグで四角く囲みます。
    *   除外したい文字があれば「除外文字」欄に入力し、「＋ 設定を追加して次へ」または「完了して閉じる」を押します。

4.  **実行:**
    画面下部の緑色のボタン「🚀 選択対象のDXFを処理してExcelに出力」を押します。保存先フォルダに統合されたExcelが出力されます。

### パターンB：図面内のすべてのテキストを一括抽出する

図面内のすべての文字を、画面上の位置関係を保ったままExcelに出力します。複数ファイル選択時は、全図面のデータが1ファイルにまとめられます。

1.  **対象の選択:**
    画面上部の「📄 DXFファイルを選択」または「📁 DXFフォルダを選択」から対象を選びます。

2.  **設定:**
    「2. 設定」欄の「2. 図面内のすべてのテキストを抽出」を選択します。
    ※ 出力結果を見て、本来同じ行になるべき文字が別の行に分かれてしまう場合は、「テキストズレ許容値」の数値を少し大きくして再度お試しください。

3.  **実行:**
    画面下部の緑色のボタン「🚀 選択対象のDXFを処理してExcelに出力」を押します。
